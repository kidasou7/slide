# PineStructar プレゼンテーション - デプロイ用パッケージ

## 📦 内容物

このパッケージには以下が含まれています：

1. **index.html** - メインのスライドショーファイル
2. **pinestructar_presentation/** - スライド画像フォルダ
3. **pinestructar_icon_square.png** - クラウドワークス用正方形アイコン

## 🚀 デプロイ方法

### GitHub Pages でのデプロイ

1. GitHubで新しいリポジトリを作成
2. このフォルダの内容をリポジトリにプッシュ：
   ```bash
   git init
   git add .
   git commit -m "Initial commit: PineStructar presentation"
   git branch -M main
   git remote add origin [your-repo-url]
   git push -u origin main
   ```
3. リポジトリの Settings → Pages → Source を "main branch" に設定
4. 数分後、`https://[username].github.io/[repo-name]/` でアクセス可能に！

### Netlify でのデプロイ

1. [Netlify](https://www.netlify.com/) にログイン
2. "Add new site" → "Deploy manually"
3. このフォルダをドラッグ&ドロップ
4. 即座に公開URL取得！

### Vercel でのデプロイ

1. [Vercel](https://vercel.com/) にログイン
2. "New Project" → GitHubリポジトリを選択
3. 自動デプロイ完了！

## 🎨 スライドの操作方法

- **矢印キー（← →）**: スライド切り替え
- **スペースキー**: 次のスライドへ
- **画面の矢印ボタン**: クリックで切り替え
- **下部のドット**: クリックで特定のスライドへジャンプ
- **スワイプ（モバイル）**: 左右スワイプで切り替え
